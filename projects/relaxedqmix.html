Overview

RelaxedQMIX is a modification of the QMIX algorithm for multi-agent reinforcement learning (MARL) in StarCraft II. By introducing a learned correction term to QMIX‚Äôs mixing network, the project relaxes the strict monotonicity constraint, enabling agents to explore strategies such as tactical retreats and sacrificial plays that standard QMIX cannot capture.

Problem

While QMIX provides strong performance in cooperative MARL tasks, its strict monotonicity constraint severely limits coordination. Agents are forced to only take actions that increase their own Q-values, discouraging short-term sacrifices that could benefit the team‚Äôs long-term success. As a result, QMIX struggles in heterogeneous and asymmetric environments.

Solution

  • Implemented a correction network with state re-injection and L2 regularization.
  • Interpolated between strict monotonicity (Œµ = 0) and flexible coordination (Œµ > 0).
  • Integrated RelaxedQMIX into the PyMARL framework and evaluated on SMAC maps.

Implementation

Experiments were conducted on BU‚Äôs Shared Computing Cluster using NVIDIA V100 GPUs and SLURM scripts.

Configuration: Œµ-greedy exploration (1.0 ‚Üí 0.05), replay buffer = 5000, target update every 200 episodes, RMSProp optimizer (lr = 0.0005).

Results

  • 8m & 2s3z: matched QMIX performance (~90% win rate).
  • MMM2: RelaxedQMIX achieved ~70% win rate vs QMIX‚Äôs 10%.
  • Corridor: remained unsolved (0%), showing the limits of monotonic relaxations.

Impact

RelaxedQMIX demonstrates that small architectural changes can significantly improve cooperative agent learning. It preserved QMIX‚Äôs strengths on simple maps while achieving substantial improvements in more complex heterogeneous scenarios.
