RelaxedQMIX: Relaxing Monotonicity Constraints in Multi-Agent Reinforcement Learning

Overview

As part of my graduate research, I extended the PyMARL framework to implement RelaxedQMIX, a modification of the QMIX algorithm for cooperative multi-agent reinforcement learning (MARL). This project aimed to relax QMIX’s monotonicity constraint to improve agent coordination.

What I Did

  • Designed and implemented the RelaxedQMIX architecture with a learned correction term.
  • Integrated the correction network into the PyMARL codebase with stabilization mechanisms.
  • Configured and ran experiments on BU’s SCC cluster using NVIDIA V100 GPUs and SLURM scripts.
  • Evaluated RelaxedQMIX against QMIX across four SMAC maps (8m, 2s3z, MMM2, corridor).
  • Analyzed win rates, training loss, and average rewards.

Results

  • Matched QMIX performance on simple maps (8m, 2s3z) with >80% win rate.
  • Achieved ~70% win rate on MMM2 vs QMIX’s <10%.
  • Showed RelaxedQMIX enabled agents to explore tactical retreats and sacrifice strategies.

Tech Stack

Python · PyTorch · PyMARL · StarCraft II (SMAC) · SLURM · HPC (BU SCC)
